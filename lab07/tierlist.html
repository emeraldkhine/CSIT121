My Reflection

In this project, I learned how to style rows with the same rank together by creating class elements. Instead of depending on something like nth-child, I can control it better by using my own classes rank-1, rank-2, etc. I also learned how to build a table and a form properly. The table part is actually not that different from data analysis. It is just organizing data in a clean and structured way, so it makes sense to the user. I learned a lot from the course links

  • MDN Web Docs
  • WebAIM
  • W3Schools
  • Learn HTML Forms by Bro Code
  • HOW TO CREATE & STYLE FORMS IN HTML by Dani Krossing

They helped me understand how to use different inputs like text, email, radio, checkbox, and range, and also how to make the form more accessible. I also used Claude Ai to explain some concepts to me in a simpler way.
