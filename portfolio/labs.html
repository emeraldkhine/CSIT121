Skills Learned This Semester

Across all ten labs, I developed the following core skills:

  1. Writing valid, semantic HTML5 using structural elements like header, main, article, section, and footer
  2. Styling pages with CSS - selectors, the box model, flexbox, grid, and CSS custom properties
  3. Building responsive layouts with @media queries for mobile and desktop
  4. Using JavaScript to interact with the DOM and fetch external data (XML, JSON)
  5. Evaluating websites for usability and accessibility using WCAG 2.2 criteria
  6. Following W3C web standards and validating HTML and CSS without errors
