A Guide to Web Graphics

Visual Reference — CSIT 121

When you see an image on a website, it is saved in a specific file format. Each format is different. Some are better for photos, some for logos, and some for illustrations. This site looks at three common formats: JPG, PNG, and SVG.

Picking the right format matters. It affects how clear your image looks, how fast your page loads, and whether your image has a transparent background. Each page below explains one format with a real example image.

.jpg JPEG Great for photos. Makes files smaller but loses a little quality. .png PNG Keeps full quality. Supports transparent backgrounds. .svg SVG Made of shapes, not pixels. Looks sharp at any size.
