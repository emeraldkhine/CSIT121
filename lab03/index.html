Lab 02 Reflection

My first webpage! At first, the process seemed daunting, with so many tags and structures to remember, but breaking it down step by step made it manageable. Learning how to properly structure an HTML document helped me understand how webpages are built and displayed.

What I Learned

  1. I learned how to structure a basic HTML page using <html>, <head>, and <body>.
  2. I learned how to use headings, paragraphs, and lists to organize content.
  3. I learned how to add links and follow web standards using trusted sources like MDN and W3Schools.

Learning about basic structure was another exciting step. I just learned how to write <html>, <head>, and <body>.

  • MDN Web Docs
  • W3Schools

Lab 03 Reflection

This lab introduced me to CSS, and I love how powerful it is for controlling the look of a webpage. Connecting an external stylesheet to my HTML file felt like easy to maintain. I could change colors, fonts, and spacing all in one place without touching the HTML structure. I found the difference between ID and class selectors. I was confused about when to use padding versus margin but I searched online and read abut it. I want to explore more CSS properties in future labs. Resources that helped me the most were MDN Web Docs and the YouTuber Dave Gray. His tutorials are clear and easy to understand.

New Things I Learned

  1. How to link an external CSS file using the <link> element in <head>.
  2. How to style elements using selectors like body, h1, p, and a.
  3. The difference between ID selectors (#) and class selectors (.).
  4. Different between padding and margin.

Additional resources I used for this lab:

  • MDN: CSS First Steps
  • CSS Tutorial by Dave Gray
